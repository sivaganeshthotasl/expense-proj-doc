**Expense Tracking Application – Overview**

**The Expense Tracking Application is a simple web-based system used to record and monitor daily expenses. It helps users keep track of how much money they spend and understand where their money is going.**

**How the Application Works:-**

**1. User Input**

**The user enters:**
         
  --> Amount (how much money was spent)
          
  --> Description (reason for spending, e.g., travel, food, shopping)

 **2. Data Submission**

--> Once the user submits the details, the application sends this data to the backend server.

**3. Data Storage**

--> The backend (Node.js application) processes the request and stores the data in the database (MySQL).

**4. Data Retrieval**

--> Whenever the user accesses the application, it fetches all stored expense records from the database.

**5. Display to User**

--> The frontend (served using Nginx) displays the records in a structured format, similar to a bank statement.

**Key Features:-**

--> Add new expense records easily

--> Store expenses securely in a database

--> View all past transactions in one place

--> Simple and user-friendly interface
